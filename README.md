# WasiSeguro

## Descripción del Proyecto

WasiSeguro es una solución digital enfocada en mejorar la seguridad urbana mediante mapas de riesgo, rutas seguras y alertas verificables.

## Segmentos Objetivo

- **Residentes**: Personas que se desplazan a diario y buscan moverse con seguridad.
- **Visitantes o Foráneos**: Usuarios que necesitan orientación confiable en zonas desconocidas.

## Principales Características

- **Mapa Interactuable de Riesgo**: Visualización actualizada de zonas con diferentes niveles de seguridad.
- **Rutas Seguras**: Cálculo de trayectos con estimación de costo por desvío.
- **Alertas Verificables**: Sistema de notificaciones validadas sobre incidentes.

- **Reportes Ciudadanos**: Plataforma colaborativa para reportar incidencias de seguridad.
